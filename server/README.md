# Derek Backend

## Requirements
- Node.js (LTS)
- MongoDB Atlas account
- OpenAI account (for GPT features)

## Setup
1. Clone repository
2. Navigate to server folder
3. Install dependencies:
   npm install
4. Create .env file using .env.example
5. Start server:
   npm start

## Features
- User authentication (JWT)
- Token-based usage system
- Prompt library (DB-managed)
- GPT API integration
- Custom prompt requests

## Notes
- MongoDB Atlas is used as content manager
- OpenAI billing is client-owned
- No admin panel required
